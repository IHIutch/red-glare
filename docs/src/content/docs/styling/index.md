Red Glare is built on USWDS's Sass theming system. Customization happens by overriding USWDS tokens in a settings file the integration picks up at build time.

## In this section

- **[Theming](/styling/theming/)** — override USWDS tokens (colors, fonts, spacing).
- **[CSS architecture](/styling/css-architecture/)** — how classes are organized (`usa-*` vs. `rg-*`, utility classes, breakpoints).

All configuration goes through the single options object passed to `redGlare()` in `astro.config.mjs`. Every option is validated by a zod schema at build time.

## In this section

- **[Overview](/configuration/overview/)** — the full option surface at a glance.
- **[Site metadata](/configuration/site-metadata/)** — title, description, logo, custom head tags.
- **[Navigation](/configuration/navigation/)** — header nav and auto-generated sidebar.
- **[Footer & identifier](/configuration/footer-identifier/)** — USWDS footer and required federal identifier strip.
- **[Government banner](/configuration/government-banner/)** — the "official website" banner and site alerts.
- **[Edit links](/configuration/edit-links/)** — "Edit this page on GitHub" links.
- **[Internationalization](/configuration/i18n/)** — default locale, locale registry, language switcher.

Red Glare's component directives let you author structured content inside plain markdown — no JSX imports, no layout gymnastics. Every component maps to USWDS markup and follows its accessibility guidance.

## In this section

- **[Alert](/components/alert/)** — USWDS alert for notices, warnings, errors, and successes.
- **[Accordion](/components/accordion/)** — expandable panels with deep-link support.
- **[Code group](/components/code-group/)** — tabbed code blocks for language or tool alternatives.
- **[Endpoint](/components/endpoint/)** — REST endpoint reference with structured parameters and examples.
- **[Link button](/components/link-button/)** — USWDS-styled call-to-action anchors.
- **[Process list](/components/process-list/)** — USWDS numbered steps with connector lines.
- **[Summary box](/components/summary-box/)** — highlight the most important content on a page.
- **[Tabs](/components/tabs/)** — generic tab panels for grouping equivalent content.

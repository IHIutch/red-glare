`astro build` produces a `dist/` directory of static HTML, CSS, JS, and assets. Any static host serves it. This site's canonical reference deployment is Cloudflare Workers.

## In this section

- **[Cloudflare Workers](/deployment/cloudflare-workers/)** — the canonical setup, with `wrangler.jsonc` config and a CI/CD example.
- **[Other static hosts](/deployment/static-hosts/)** — Netlify, Vercel, GitHub Pages, S3, Azure Static Web Apps.

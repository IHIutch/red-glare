Beyond components and layouts, Red Glare ships a handful of opinionated defaults that make docs sites production-ready out of the box.

## In this section

- **[Search](/features/search/)** — full-text Pagefind integration.
- **[RSS](/features/rss/)** — auto-generated `feed.xml`.
- **[llms.txt](/features/llms-txt/)** — AI-accessibility endpoints following [llmstxt.org](https://llmstxt.org).
- **[Link validator](/features/link-validator/)** — build-time internal link validation.
- **[TOC, breadcrumbs, prev/next](/features/toc-breadcrumbs/)** — automatic in-page navigation chrome.
- **[Accessibility](/features/accessibility/)** — USWDS WCAG 2.1 AA baseline and what you still need to get right.
